# Metrics Dashboard

Custom Metrics Dashboard for a Ford initiative to analyze customer engagement with SMS and email service appointment reminders. This tool automated a previously manual, multi-person workflow by parsing CSV database exports, instantly calculating key performance metrics, and generating structured reports for immediate export.

### Tooling

* Frontend: React - [TypeScript/JavaScript]
* Build Tool: [Vite]
* Styling: [Vanilla CSS]
* CSV Parser: [PapaParse]
